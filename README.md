# JavaSwingChatGPT
Java Swing으로 ChatGPT API연결해서 사용하는 프로그램입니다.

만든 이유. 웹상에서는 프로그래밍 코드입력을 많이 할수가 없어서 만드는 군요. 아직, FileUpload구현은 하지 않았습니다. 아마 스트링 파일로 전송이 될까... 생각하는군요.
